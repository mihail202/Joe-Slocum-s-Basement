Описание

Небольшое двухуровневое плохо освещённое помещение в подвале Кафе «Слокам Джо». На верхнем уровне расположена только лестница и дверь из кафе, на нижнем — кухня для приготовления блюд с плитами в несколько рядов и небольшим количеством хлама.

Также здесь есть лифт, замаскированный отворачивающимся в сторону шкафом, ведущий в Коммутатор. Однако до выполнения квеста «Агентурная работа» этот лифт нельзя активировать (потребуется терминал в Коммутаторе прям у выхода из лифта).

Через эту локацию происходит выход из Коммутатора в квесте «Агентурная работа» в лифте, не сопровождающийся появлением загрузочного экрана, поэтому данную локацию можно идентифицировать только при входе со стороны кафе.

Агентурная работа

Появление

Подвал Джо Слокама появляется только в Fallout 4.
